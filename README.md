# Spring Boot Project
# drone-delivery


## Project Description

This Spring Boot application is a RESTful API for drone deliveries. It supports basic CRUD operations for registering drones and adding medication load.

## Technologies Used
- Spring Boot
- Spring Data JPA
- H2
- Lombok

## Installation

### Prerequisites

- Maven
- H2 database
